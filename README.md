TalentHub – Job Board Web Application

A simple Job Board web application built using Flask and SQLite.
This platform allows Employers to post jobs and Candidates to apply with their resumes.

🚀 Features
User authentication (Candidate / Employer)
Role-based dashboards:
Employer Dashboard – Add, manage, and view job postings
Candidate Dashboard – View jobs, apply with User ID, Mobile Number, and Resume
Job search functionality
Responsive UI for desktop and mobile
Display job details including:
Salary
Joining Date
Qualifications
Experience
Footer with website info (Phone, Email, Instagram, LinkedIn, Facebook)

🛠️ Tech Stack
Backend: Python (Flask)
Database: SQLite
Frontend: HTML, CSS
Version Control: Git & GitHub
